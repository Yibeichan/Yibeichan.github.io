正则表达式最常用的是search（）和findall（）

本周题目

The basic outline of this problem is to read the file, look for integers using the re.findall(), looking for a regular expression of ‘[0-9]+’ and then converting the extracted strings to integers and summing up the integers.

http://python-data.dr-chuck.net/regex_sum_291515.txt (There are 78 values and the sum ends with 813)

答案

备注

两个问题：

1）第一次split后出现列表嵌套不知道该怎么办了，差点用上了字典，后来翻到《列表》的作业才想起来用append加到另一个列表。

2）表达式出错了，第一次用的是[0-9]+？，也不知道为什么在这用了一个非贪婪模式，是担心会把后面的单词也写进来了吗？真是杞人忧天。然后又发现’’这个单引号没加。输出之后发现变成了单个的数字，于是又在？后加了一个空格。本以为这下做对了，但是求和发现和答案不符，对着txt比对半天，原来是把一行结尾处的数字落下了，因为那个数字后面是没有空格的。好吧，空格和？都去掉了，结果竟然对了，之前真是瞎折腾。

Cheat Sheet :)

表达式 含义
^ Matches the beginning of a line
$ Matches the end of the line
. Matches any character
\s Matches whitespace
\S Matches any non-whitespace character
* Repeats a character zero or more times
*? Repeats a character zero or more times (non-greedy)
+ Repeats a character one or more times
+? Repeats a character one or more times (non-greedy)
[aeiou] Matches a single character in the listed set
[^XYZ] Matches a single character not in the listed set
[a-z0-9] The set of characters can include a range
( Indicates where string extraction is to start
) Indicates where string extraction is to end
